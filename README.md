Gratin
===

Machine Learning Tool in [Groovy](http://www.groovy-lang.org/)  
Deep learning will be covered in near future. 

##Networks,Layers

- Hopfield Network
- Boltzmann Machine (simple, without hidden units, fully connected)
- RBM (Restricted Boltzmann Machine)
- Softmax Layer

##Utilities

- Matrix

##Tests
